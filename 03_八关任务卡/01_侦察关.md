# 第 1 关 · 侦察关(W1–2)

## 目标
搞清楚两件事:市场在造什么(产品),市场在买什么(岗位)。产出你的**个人赛道声明**——后面七关的所有项目都为这条赛道服务。

## 任务
1. **拆 3 个 AI 产品**(至少 1 个国产、1 个 Agent 形态,如:Kimi/豆包、Cursor、Manus、NotebookLM):用户是谁 / 解决什么 / 技术路线猜测(模型+RAG?+工具?)/ 收费逻辑 / 我能复刻的最小切片
2. **拆 10 份真实 JD**(BOSS直聘/国家大学生就业平台搜"AI 应用开发/大模型应用/Agent 工程师"):把技能要求逐条抄进表格,统计词频
3. **写个人赛道声明**(半页):我瞄准的岗位 × 行业场景 × 我的差异化(专业背景/数据资源/兴趣)

## 验收 rubric
- [ ] JD 词频表里出现:MCP / 评测 / RAG / Agent / 可观测(说明你找的是真 JD 不是老 JD)
- [ ] 每个产品拆解含"我能复刻的最小切片",且切片 ≤2 周工作量
- [ ] 赛道声明能回答:"为什么是你做这个方向?"

## 简历句模板
> (本关不直接产简历句,产**简历的靶心**——之后每关的简历句都朝它写)

## 面试追问预演
- 你为什么选这个方向?(答不出差异化 = 重写声明)
- 这个产品如果让你做,你先砍掉什么功能?

## 双工具用法
- **NotebookLM**:把 10 份 JD 全文上传 → 生成报告"共性技能要求 Top10" → 生成测验自查认知盲区
- **Obsidian**:建 `赛道/` 目录,JD 表格用 Bases 建表(属性:公司/岗位/技能词/薪资),自动聚合词频
