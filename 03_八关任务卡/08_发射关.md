# 第 8 关 · 发射关(W15–16)

## 目标
把七关的积累组装成**应聘弹药库**,并通过模拟终面压力测试。

## 2026 简历筛选事实
- 花哨 side project 无效;**干净 repo + eval traces + 成本记录 + 失败复盘** = 直通二面
- GitHub 开源参与/高星可跳初筛;字节考"场景化编程",美团直接看 GitHub

## 任务
1. **GitHub 作品集**:主项目 repo 达到"陌生工程师 10 分钟能跑起来"标准——README(问题/架构图/quickstart/评测结果表/成本表)、eval 报告入库、复盘文档链接
2. **简历**:项目区 3 条,每条 = 七关攒的简历句(动词开头+技术栈+量化结果);用第 1 关 JD 词频表反查覆盖率
3. **2 分钟路演**:问题→演示→架构一句话→评测数字→下一步;录视频,NotebookLM 转录后自查废话率
4. **模拟终面**:教练智能体 4 人格轮番 20 分钟 + 同学互面 1 轮 + 教师抽面
5. **失败复盘集**:从 Obsidian `#failure-mode` 笔记里选 3 个最疼的,写成"我如何搞砸又修好"的故事——高级面试官最爱

## 验收 rubric
- [ ] 找一个外班同学实测:10 分钟内跑起你的项目
- [ ] 简历技能词覆盖目标 JD 词频表 Top10 中 ≥7 个,且每个词有项目证据支撑(无证据的词删掉)
- [ ] 模拟终面 4 人格全部"通过"或修复后复面通过
- [ ] 路演视频 ≤2 分钟,含至少 2 个量化数字

## 面试追问预演(终面高频)
- 整个学期你最难的技术决策是什么?
- 如果重做一次,你会改什么架构?
- 你的项目和市面上 XX 产品比,凭什么?

## 双工具用法
- **NotebookLM**:把你全部复盘+评测报告建一个"我的面试笔记本"→ 用**互动音频模式**通勤时被 AI 主持人追问,练即兴表达;闪卡刷八股与自己项目细节
- **Obsidian**:作战室首页放"发射检查清单" Bases 看板;面试后每场写战报(被问什么/卡在哪),错题进第 5 关做的 MCP 工具自动归档——**闭环了**
