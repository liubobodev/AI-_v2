# 启动指南

## 方式一：本机使用

双击桌面 **启动AI教练.command** → 自动打开 http://localhost:3000

## 方式二：外网访问（电脑 + 手机）

双击桌面 **启动AI教练_外网.command** → 获得公网地址，任何设备可访问。

首次使用需注册 ngrok 并配置 authtoken（见桌面文件内说明）。

## 方式三：永久部署到 Vercel（推荐 · 免费）

**效果**：得到 `https://xxx.vercel.app` 固定地址，24 小时在线，电脑手机都能用，本地电脑关机也能访问。

### 部署步骤（5 分钟）

1. 把本文件夹 push 到 GitHub 仓库
2. 打开 https://vercel.com ，用 GitHub 登录
3. 点「Import」→ 选择你的仓库
4. **关键设置**：
   - Framework：Next.js（自动识别）
   - Root Directory：留空（从仓库根目录构建）
   - Build Command：`cd webapp && npm install && npm run build`
   - Output Directory：`webapp/.next`
   - Install Command：`cd webapp && npm install`
5. 环境变量添加：`DEEPSEEK_API_KEY` = `sk-bc6ba2b71e524c558494d6d264630619`
6. 点 Deploy，等 2 分钟完成
7. 得到 `https://xxx.vercel.app`，发给学生即可

### 更新知识库后重新部署

修改了 Markdown 文件后，push 到 GitHub → Vercel 自动重新部署（约 2 分钟）。

### Vercel 限制（免费版）

- 每月 100GB 带宽（个人+小班足够）
- Serverless 函数每次最长 10 秒（DeepSeek 流式响应走 streaming，不受此限）
- 知识库已编译进代码，无需额外存储

## 教师面板

访问 `http://localhost:3000/teacher`（外网时对应 ngrok/Vercel 地址 `/teacher`）

- 用管理员创建的教师账号登录
- 查看全班学生的关卡进度、技能评估、训练次数
- 红色标记卡关学生，一键定位需要帮助的人

## 管理员后台

访问 `http://localhost:3000/admin`

- 默认密码：`admin123`（可在 `.env.local` 的 `ADMIN_PASSWORD` 修改）
- 创建/删除教师和学生账号
- 查看所有用户列表和登录记录
