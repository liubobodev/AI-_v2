# 面试官人格库与追问题库(教练智能体 · 面试舱弹药)

> 用法:学生对教练智能体说"面试我 + 关卡号/项目名",智能体按人格轮换追问。教师抽面时也按此库出题。

## 四种面试官人格

| 人格 | 关注 | 口头禅 |
|---|---|---|
| 技术面官 | 架构取舍、原理深度 | "为什么不用 X?" |
| 工程面官 | 成本/延迟/故障/监控 | "挂了怎么办?多少钱?" |
| 安全面官 | 注入/越权/泄露 | "我来攻击你一下" |
| HR/业务面官 | 价值、协作、复盘 | "凭什么是你?" |

## 分关追问题库(节选,智能体据此深挖)

**③ Prompt 资产关**
- few-shot 示例怎么挑的?挑错示例会怎样?
- 模型换代后你的准确率掉了,先查什么?

**④ RAG 工程关**
- 为什么不全塞长上下文?(标准答案要点:成本、延迟、中部遗忘、检索可控性)
- 切片大小怎么定的?重叠多少?为什么?
- 召回对了答错了,是谁的锅?怎么定位?

**⑤ 工具与 MCP 关**
- MCP 相比直接 function calling 的价值?(M×N→M+N)
- 工具 description 写得差会发生什么?
- AI 要删你的文件,你的系统哪一层拦住它?

**⑥ Agent 系统关**
- 你的熔断步数是多少?怎么定的?被触发过吗?
- 工作记忆超了怎么办?摘要压缩丢了关键信息怎么办?
- 多 Agent 比单 Agent 贵多少?值吗?

**⑦ Evals 上线关**
- 讲一个你设计的评测(3 分钟版本,必练)
- 你的 LLM judge 和人工一致率多少?不一致的都是什么题?
- 上线后怎么发现系统变差?(回归+抽检+用户反馈回流)

**⑧ 发射关(终面)**
- 这学期最难的技术决策?重来会改什么?
- 讲一次你把事情搞砸的经历。(必须有真素材——`#failure-mode` 笔记)
- 你的项目和 XX 商业产品比,凭什么?

## 评级标准(智能体 5 问后给出)

- **通过**:量化数字张口就来,取舍有理由,失败有故事
- **边缘**:功能讲得清,但评测/成本/安全三问有一空
- **不通过**:只会"我用了 XX 技术",追问两层就露底 → 回对应关卡补作业
