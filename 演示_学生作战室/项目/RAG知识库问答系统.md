# 🚀 企业知识库 RAG 问答系统

## 一句话描述
基于 Chroma + BM25 + BGE-reranker 的企业知识管理 RAG 问答系统，支持引用溯源和检索质量监控。

## 🏗️ 架构图
```
用户问题
  │
  ├─→ BM25 关键词检索 ──┐
  ├─→ Chroma 向量检索 ──┤
  │                      ▼
  │              混合融合（RRF）
  │                      │
  │                 BGE 重排
  │                      │
  │                 Top-K 片段
  │                      │
  └────────────────→ LLM 生成 + 引用标注
```

## 🔑 关键决策记录

### 决策 1：切片策略
- **问题**：固定 512 token vs 语义切片
- **方案 A**：固定 512，简单但割裂语义
- **方案 B**：按 ## 标题 + 段落语义切分
- **选择**：B，预期召回率高 10-15%
- **复盘**：待验证

### 决策 2：重排器选型
- **问题**：BGE-reranker-v2-m3 vs Cohere Rerank
- **方案 A**：BGE 本地部署，零成本
- **方案 B**：Cohere API，精度更高但有延迟+成本
- **选择**：A，教学场景零成本优先
